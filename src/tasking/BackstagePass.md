    * 5 < 保质期 < 11：
        * 价值 = 49：过一天，价值 = 50
        * 价值 < 49：过一天，价值 +2

        * 价值 = 48：过一天，价值 = 50
        * 价值 < 48：过一天，价值 +3
        * 价值 = 50：过一天，价值不变
2. 超出保质期（SellIn <= 0）  
    过一天，价值均为 0
